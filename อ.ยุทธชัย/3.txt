php framework การรจัดการสำหรับฐานข้อมูล อ้างอิงจากการพัฒนารูปแบบ MVC


โดย Chanchai Supaartagorn

php เป็นภาษาฝั่ง Server ที่มีความสารมารถอย่างมากสำหรับการพัฒนา Web Applocation ทั้งในเรื่องของประสิทธิภาพ การขยายระบบ และยังเป็น Open source

สิ่งหนึงที่เป็นปัญหาของนักพัฒนาในการพัฒนา Web Application คือการพัฒนาในรูปแบบของ  dynamic website ,web application,web service 
นักพัฒนาต่างพัฒนาไปในแนวทางของตัวเองไม่มีมารตฐานที่แน่นอน

การนำรูปแบบการพัฒนา Web Application โดยใช้ หลักการของ MVC คือ Model,View,Controller ทำให้เกิดเป็นมารตฐานขึ้นใหม่ในการพัฒนาและแบ่งแยกการพัฒนาได้ชัดเจน
และยังเพิ่ม library ที่เป็นมาตรฐาน เช่น active record ในการจัดการฐานข้อมูล
